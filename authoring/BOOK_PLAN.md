# Book Plan — Think Python (thinkpy.org)

## Title and Publication

- **Title**: Think Python
- **URL**: thinkpy.org
- **Format**: Jupyter Book (open access, online)
- **Inspired by**: Think Python, 3rd edition, by Allen B. Downey

## Audience

- College students in an intermediate to advanced Python programming course
- Prerequisites: basic programming experience assumed
- Application context: Python programming, data science

## Goals

- Teach Python with CS concepts as a priority
- Use data science as the applied context throughout
- Prepare students for technical interviews
- Stay current with industry practices
- Some math is welcome, especially when it clarifies a concept

## Scope

Four parts:

| Part | Topic           | Chapters                                                                    |
|------|-----------------|-----------------------------------------------------------------------------|
| I    | Fundamentals    | 01 Intro, 02 Basics, 03 Control Flow, 04 Functions, 05 Exceptions & Testing |
| II   | Data Structures | 06 Collections, 07 Dictionaries, 08 Strings and Text                        |
| III  | Program Design  | 09 OOP, 10 Functional Programming, 11 Iterators & Generators, 12 APIs       |
| IV   | DSA             | 13 Abstract Data Structures, 14 Algorithms                                  |

Appendices cover tooling, Jupyter setup, modules/packaging, and bibliography.
